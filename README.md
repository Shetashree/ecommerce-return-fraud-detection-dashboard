# 🛒 E-Commerce Return Fraud Detection Dashboard

An interactive Power BI dashboard built using Snowflake SQL and Power BI to identify return fraud patterns, suspicious customers, high-risk categories, and state-wise return behavior.

---

## Project Overview

This project analyzes e-commerce return transactions to uncover fraud trends and customer behavior.

The dashboard enables business teams to:

- Monitor total returns
- Track fraud loss
- Identify suspicious customers
- Compare return value across states
- Analyze category-wise return trends
- Monitor return windows

## Key Insights

- Beauty category records the highest number of returns.
- Return value varies significantly across states.
- Certain customers exhibit unusually high return rates.
- Fraud loss is concentrated in a few product categories.
- Most returns occur within the first week after delivery.
